# Крестики-нолики (NCurses)

Классическая игра «Крестики-нолики» с терминальным графическим интерфейсом на базе библиотеки NCurses.

## Возможности

- Игровое поле 3×3
- Пошаговая игра для двух игроков (X и O)
- Управление с клавиатуры
- Подсветка выбранной ячейки
- Определение победителя по горизонталям, вертикалям и диагоналям
- Чистая сетка с использованием символов псевдографики Unicode

## Управление

| Клавиша | Действие |
|---------|----------|
| Стрелки | Перемещение курсора |
| Enter | Поставить символ (X или O) |
| Q | Выйти из игры |
